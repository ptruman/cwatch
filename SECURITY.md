# Security Policy

## Supported Versions

The initial upload to GitHub and Docker was V4.  Support began at V4.

| Version | Supported          |
| ------- | ------------------ |
| 5.0.x   | :white_check_mark: |
| 4.0.x   | :x: |
| < 4.0   | :x:                |

## Reporting a Vulnerability

Raise an issue via GitHub if you suspect a vulnerability.
